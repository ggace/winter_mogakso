## 1) 문제 설명
숫자 n이 주어졌을때, 재귀로 푸는 코드와 dp를 사용해 푼 코드의 실행 횟수를 출력하는 문제이다.

## 2) 문제 풀이
재귀로 푸는 코드는 n번째 피보나치 수만큼 계산을 해야한다. 하지만, dp를 사용한 코드는 n번의 반복이 필요하다.

## 3) 코드
```python
n = int(input())

dp = [0, 1, 1]

for i in range(3, n+1):
    dp.append(dp[i-1] + dp[i-2]);

print(f"{dp[n]} {n-2}")
```
